## A. 垃圾詞條（必刪）— Groq 自動補詞誤建的片語碎片

| 詞條 | 問題 |
|---|---|
| `萬軍之耶和華` | 片語，非專名 |
| `華以色列` | 「耶和**華**+**以色列**」相鄰被切 |
| `色列人` | 「以**色列人**」碎片 |
| `耶和華的靈` | 片語 |
| `巴比倫王尼` | 「**巴比倫王尼**布甲尼撒」被切，導致亂框 |
| `主耶和華` | 片語（耶和華已有詞條） |
| `撒冷` / `路撒冷` | 「耶**路撒冷**」碎片 |

→ 刪除，避免亂框。

## B. 通用常識名詞（建議降級/刪）— 違反「只標艱澀專名」原則

`僕人`、`百姓`、`居民`、`城邑`、`首領`、`誡命` — 白話常用詞，標了變大鍋菜，稀釋標註價值。

## C. 同名不同人未消歧義（最高風險 — 「一亂全毀」核心）

| 詞條 | 現況 | 實際對應 |
|---|---|---|
| `馬利亞` | 只指耶穌之母 | ①耶穌之母 ②抹大拉 ③伯大尼(馬大之妹) ④雅各之母 ⑤馬可之母 |
| `雅各` | 單一 | ①OT族長 ②門徒(西庇太子) ③小雅各 ④主的兄弟 |
| `腓力` | 單一 | ①門徒 ②執事(傳福音) ③分封王希律腓力 |
| `撒迦利亞` | 單一 | ①小先知 ②施洗約翰之父 ③被殺祭司 |
| `拿單` | 單一 | ①先知 ②大衛之子 |
| `約阿施` | 單一 | ①猶大王 ②以色列王 ③基甸之父 |
| `希律` | 有「非特指」說明但單卡 | 大希律/安提帕/亞基帕 |

已正確消歧義：`西門`、`猶大`、`亞拿尼亞`、`耶羅波安`、`約翰`(施洗轉址)
